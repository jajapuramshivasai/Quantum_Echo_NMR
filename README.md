## Setup Instructions

1. Create a Python virtual environment:
    ```bash
    python3 -m venv .venv
    ```

2. Activate the virtual environment:
    - On macOS/Linux:
      ```bash
      source .venv/bin/activate
      ```
    - On Windows:
      ```bash
      .venv\Scripts\activate
      ```

3. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```
